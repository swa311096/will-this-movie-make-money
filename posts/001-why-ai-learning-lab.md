# Why I Started AI Learning Lab

I wanted one place where AI concepts and working code live together.

Most AI learning online is fragmented. You read about prompts in one place, RAG somewhere else, agents in another thread, and evals almost as an afterthought.

So I started `ai-learning-lab` with one rule:

learn the concept, build it, verify it, then explain it clearly.

The goal is to understand and implement:

- LLM basics
- structured outputs
- RAG
- tool calling
- agents
- MCP
- skills and agent instructions
- evals

The idea is simple: if I cannot build it and explain it, I do not understand it well enough yet.
