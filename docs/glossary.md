# Glossary

This file is a short reference for keywords used across the repository.

## Agent

A system in which the model decides what to do next across multiple steps.

## Context Window

The amount of input text the model can consider in one request.

## Eval

A test or measurement for AI system behavior.

## LLM

Large Language Model.

## MCP

Model Context Protocol, a standard way to connect models with tools and resources.

## Prompt

The instructions and input sent to the model.

## RAG

Retrieval-Augmented Generation: retrieve relevant information first, then generate an answer using it.

## Retrieval

The process of finding useful external information and bringing it into the model's input.

## Skill

A reusable instruction set for a specific kind of agent task, usually documented in a `SKILL.md` file.

## Structured Output

Machine-readable output from the model, such as JSON.

## Tool

A callable function or capability outside the model that the system can use.

## Workflow

A mostly fixed sequence of steps controlled by code.
