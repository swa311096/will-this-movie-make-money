# Part 1: Before Transformers

This part explains the path from early language-processing systems to the ideas that made transformers possible.

The main mistake people make here is moving too quickly.

If you skip this part, modern terms like:

- context
- embeddings
- attention
- sequence models

start sounding like random jargon instead of answers to earlier problems.

## What This Part Covers

1. why language is hard for machines
2. why rule-based systems hit limits
3. how statistical NLP improved things
4. why embeddings mattered
5. how sequence models and attention changed the game

## How To Read This Part

Read slowly and ask:

- what problem was this method solving
- what did it do better than the previous method
- what weakness remained

That pattern is the real history.
